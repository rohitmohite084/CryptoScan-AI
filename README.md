## 📌 Project Overview
This project is an advanced cryptocurrency market trading terminal designed to analyze real-time market data and forecast price trends using Artificial Intelligence. It showcases a complete end-to-end Data Science lifecycle, from data acquisition to model deployment.



## 🛠️ Complete Project Workflow
I have followed a structured professional workflow to execute this project:

1. **Objective & Goal:** Identifying market trends to predict future cryptocurrency price movements accurately.
2. **Data Collection:** Leveraging `yfinance` API to retrieve OHLCV (Open, High, Low, Close, Volume) data.
3. **Data Preprocessing:** Performing data cleaning, feature scaling, and time-series formatting.
4. **EDA (Exploratory Data Analysis):** Analyzing statistical trends and patterns in market behavior.
5. **Forecasting Models:** Building and comparing multiple models:
    * ARIMA & SARIMA (Statistical Analysis)
    * PROPHET (Additive Time-Series Model)
    * **LSTM (Deep Learning - Final Selection for Deployment)**
6. **Model Evaluation:** Using RMSE and MAE metrics to validate model accuracy.
7. **Visualizations:** Creating interactive charts using Plotly and Matplotlib for deep insights.
8. **Deployment:** Deploying the **CryptoScan** dashboard using **Streamlit**.
